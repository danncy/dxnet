# dxnet
make it simple!
make the network, protocol virtualization and easily to understand!

a c++ project for network virtualize in linux.

* how to build:
  in linux, first you need install cmake.   
  apt install cmake   

  then,    
  cd dxnet   
  mkdir build   
  cd build   
  cmake ..    
  make   

also, in windows, you can install cygwin to work.
